Hands On Tutorials

The directory HandsOnTutorial contains a set of scripted hands-on tutorials showing how to use an MBrace cluster from F# Interactive.

  1. Provision your cluster, see mbrace.io.
  2. Open and build the solution MBrace.StarterKit.sln before working with the scripts to restore the necessary packages.
